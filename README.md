Este projeto tem como objetivo centralizar todos os fluxos que já criei utilizando o **n8n**.

Grande parte dos workflows armazenados aqui foi desenvolvida para **automação de marketing**.

## 🎯 Para quem é este repositório

Aqui você vai encontrar fluxos que podem aumentar sua produtividade, especialmente se você atua com marketing, como:

- gestor de tráfego
- social media
- dono(a) de agência de marketing
- profissionais da área que querem automatizar processos

Se você trabalha com marketing, estes fluxos podem te ajudar em diferentes etapas da operação.
